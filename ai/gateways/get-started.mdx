If you're exploring the possibility of joining the AI Subnet as an _Gateway_ to
perform **AI inference task routing** on the Livepeer network provide inference
services to customers, this guide is for you. It's tailored to help you navigate
the setup process of an AI Gateway, mirroring the
[Gateway Setup Guide](/gateways/guides/gateway-overview) for the _Mainnet
Transcoding Network_, but with adjustments for AI operations. Dive into the
subpages for a step-by-step walkthrough. For a deeper understanding of Gateway
operations, the [Gateway Setup Guide](/gateways/guides/gateway-overview) is your
go-to resource.

## Prerequisites

Before you start seting up your _AI Gateway_, ensure you have:

- A [Linux](https://ubuntu.com/download/desktop) system (Support for Windows and
  MacOS coming soon)
- Root user access to the system
