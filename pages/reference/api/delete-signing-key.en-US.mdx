# `DELETE` signing key

`DELETE /access-control/signing-key/{KEY_ID}` deletes an existing `signing key` object.

## Request

```bash
curl https://livepeer.studio/api/access-control/signing-key/${KEY_ID} \
--header 'Authorization: Bearer {api_key}' \
  -X DELETE \
  -H "Content-Type: application/json" \
```

## Response

A `204 No Content` status response indicates the `signing key` was successfully deleted.
