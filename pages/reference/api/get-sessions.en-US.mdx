# Retrieve Sessions

Use `/session` to a return list of sessions with the same `userId`. This will
return a list of sessions created by the same user.

```bash
curl -H 'authorization: Bearer {api-key}' \
"https://livepeer.studio/api/session"
```

Use `/stream/{parentId}/sessions` to a return list of sessions with the same
`parentId`

```bash
curl -H 'authorization: Bearer {api-key}' \
"https://livepeer.studio/api/stream/{parentId}/sessions"
```
