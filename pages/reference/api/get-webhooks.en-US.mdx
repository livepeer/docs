# Retrieve a Webhook

You can retrieve the webhooks by calling the `GET` webhook endpoint with the `webhookId`.

### Request

```bash
curl 'https://livepeer.studio/api/webhook/{id}'
```

# Retrieve subscribed webhooks

You can retrieve webhooks subscribed to a specific event by calling the `GET` webhook subscribed endpoint with the `event` name.

### Request

```bash
curl 'https://livepeer.studio/api/webhook/subscribed/{event}'
```
